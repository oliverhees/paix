# PAIONE Dokumentation

**The personal AI that evolves with you.**

Willkommen in der PAIONE-Dokumentation. Hier findest du alles, um deinen persoenlichen KI-Assistenten zu installieren, konfigurieren und anzupassen.

## Was ist PAIONE?

PAIONE ist eine Open-Source Personal AI, die mit dir waechst. Sie lernt deine Vorlieben, merkt sich deinen Kontext und arbeitet eigenstaendig in deinem Sinne -- mit Skills, Workflows, Werkzeugen und einem semantischen Gedaechtnis.

## Die wichtigsten Features

| Feature | Beschreibung |
|---------|-------------|
| **Graphiti-Wissensgraph** | Semantisches Gedaechtnis, das unbegrenzt waechst |
| **TELOS-Identitaetssystem** | 10 Dimensionen, die deinem Assistenten beibringen wer du bist |
| **Skills** | Wiederverwendbare KI-Faehigkeiten mit Tools, Parametern und Verkettung |
| **Workflows** | Geplante, autonome KI-Aufgaben mit Chains, Webhooks und Templates |
| **Werkzeuge** | MCP Server und REST APIs als Chat-Tools verbinden |
| **Intelligentes Model-Routing** | Waehlt automatisch Haiku/Sonnet/Opus, spart bis zu 90% |
| **Multi-Kanal** | Web-UI, Telegram (weitere folgen) |
| **Self-Hosted** | Deine Daten bleiben auf deinem Server |
| **Setup-Wizard** | In 3 Schritten startklar -- Profil, API Key, fertig |
