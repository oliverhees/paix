# PAIONE Dokumentation

**The personal AI that evolves with you.**

Willkommen in der PAIONE-Dokumentation. Hier findest du alles, um deinen persönlichen KI-Assistenten zu installieren, konfigurieren und anzupassen.

## Was ist PAIONE?

PAIONE ist eine Open-Source Personal AI, die mit dir wächst — sie lernt deine Vorlieben, merkt sich deinen Kontext und arbeitet eigenständig in deinem Sinne.

## Die wichtigsten Features

| Feature | Beschreibung |
|---------|-------------|
| **Graphiti-Wissensgraph** | Semantisches Gedächtnis, das unbegrenzt wächst |
| **TELOS-Identitätssystem** | 10 Dimensionen, die deinem Assistenten beibringen wer du bist |
| **Intelligentes Model-Routing** | Wählt automatisch Haiku/Sonnet/Opus, spart bis zu 90% |
| **Autonome Hands** | Agenten, die nach Zeitplan arbeiten |
| **Multi-Kanal** | Web-UI, WhatsApp, Telegram, Discord, Slack |
| **Self-Hosted** | Deine Daten bleiben auf deinem Server |
