# Welcome to PAIONE!

Hey there! You just found your new favorite AI assistant. Let's get you set up — it's easier than you think.

## What is PAIONE?

Imagine having a super-smart assistant who actually **remembers** your conversations, knows your goals, and gets better every day. That's PAIONE.

Most AI chatbots forget everything the moment you close the tab. PAIONE doesn't. It has a real memory — like a colleague who's been working with you for months.

## What You'll Need

Don't worry, this list is short:

- **A computer** (Windows, Mac, or Linux — all work)
- **Docker Desktop** (a free program that runs PAIONE — we'll install it together)
- **An Anthropic API key** (this is like a password that lets PAIONE talk to the AI brain — we'll get one in the next step)
- ⏱️ About **10 minutes** of your time

> **"What's Docker?"** Think of it like a shipping container for software. Instead of installing a bunch of complicated stuff on your computer, Docker puts everything PAIONE needs in one neat package. You install Docker, and it handles the rest.

> **"What's an API key?"** It's basically a special password. Anthropic (the company that makes the AI) gives you this password so PAIONE can use their AI. Without it, PAIONE would be smart but... silent.
