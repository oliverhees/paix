# PAIONE Documentation

**The personal AI that evolves with you.**

Welcome to the PAIONE documentation. Here you'll find everything you need to install, configure, and customize your personal AI assistant.

## What is PAIONE?

PAIONE is an open-source personal AI that learns your preferences, remembers your context, and works autonomously on your behalf. It combines a temporal knowledge graph, intelligent model routing, skills, workflows, and a web interface anyone can use.

## Key Features

| Feature | Description |
|---------|------------|
| **Graphiti Knowledge Graph** | Semantic, temporal memory that scales without bloating context |
| **TELOS Identity System** | 10 dimensions that teach your assistant who you are |
| **Skills** | Reusable AI capabilities with tools, parameters, and chaining |
| **Workflows** | Scheduled, autonomous AI tasks with chains, webhooks, and templates |
| **Werkzeuge** | Connect MCP servers and REST APIs as chat tools |
| **Smart Model Routing** | Auto-picks Haiku/Sonnet/Opus per task, saves up to 90% |
| **Multi-Channel** | Web UI, Telegram (more coming) |
| **Self-Hosted** | Your data stays on your infrastructure |
| **Setup Wizard** | Get started in 3 steps -- profile, API key, done |
